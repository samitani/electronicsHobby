# What this ?
This is a sample code to run a stepper motor written in tinygo.
Push the button M5Stamp, then Stepper motor runs.

# Parts

| Parts            | Name                 |
|------------------|----------------------|
| Motor Driver     | Toshiba TB6674PG x 1 |
| Micro Controller | M5Stamp C3U      x 1 |
| Stepper Motor    | MDP-35A          x 1 |
